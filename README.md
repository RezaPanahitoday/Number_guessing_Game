## Description:
**This project is a simple yet fun Number Guessing Game written in Python.
The goal of the game is for the player to guess a number randomly chosen by the computer.**

## ğŸ“ Features

**Random Number Generation:**

The user defines a range, and the program picks a random number within that range.

**Input Validation:**

Only numeric inputs are accepted.

If a user enters a non-numeric value, an error message is shown and the program continues safely.

**Guess Guidance:**

After each guess, the program tells the user if the guess is too high or too low, helping them reach the correct number.

**Guess Counter:**

The program counts how many guesses the player made and displays it at the end.

**Timer:**

Measures the time spent guessing and displays it with two decimal places.

**Motivational Message:**

After finishing the game, a motivational quote is displayed to make the experience enjoyable.

## ğŸ’» How to Run

Run the program in Python 3.

Enter your desired number range.

Guess the number; the game will guide you if your guess is too high or too low.

When the correct number is guessed, the game shows the number of attempts, elapsed time, and a motivational message.
